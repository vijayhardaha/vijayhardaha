# Project README

This is a backup vault for configuration files used across Vijay's development environment.

## Directory Structure

```
/
├── configs/                    # Main config backup directory
│   ├── NextJS/               # Next.js project template configs
│   └── Parcel/               # Parcel bundler configs (future)
├── .github/                   # GitHub-specific configs
│   └── copilot-instructions.md
├── .gitignore                # Root ignore rules
├── AGENTS.md                 # Agentic coding guidelines
├── PROJECT-README.md         # This file
└── package.json              # Root package.json (formatting only)
```

## Purpose

- **Backup & Restore**: Copy/download configs to new projects for consistency
- **Not Runnable**: These are backup files, not a runnable application
- **Documentation-Rich**: Every config option is documented with comments

## Quick Start

```bash
pnpm install
pnpm format
```

## Available Config Templates

### NextJS (configs/NextJS/)

Complete Next.js project configuration - see README.md inside that folder.
